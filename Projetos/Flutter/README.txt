
-> Para rodar os códigos desta pasta basta ter o Flutter SDK e suas dependencias instalados
e realizar os seguintes passos:

1- criar um novo projeto Flutter com o comando: flutter create <nome_do_projeto>
2- entrar na pasta criada do projeto com: cd <nome_do_projeto>
3- acessar a pasta lib do projeto e apagar a main.dart padrao
4- substituir com os arquivos main.dart e outros que estão nessa pasta do git
5- voltar para o inicio da pasta do projeto <nome_do_projeto>
6- rodar o projeto com seu editor de texto ou com o comando: flutter run
